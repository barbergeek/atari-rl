# atari-rl

Atari Reinforcement Learning for Frogger  
Based on PyTorch [official tutorial](https://pytorch.org/tutorials/intermediate/mario_rl_tutorial.html) to build an AI-powered Mario.

Note: this is the baseline project. The updated code is in atari-rl-sb.

## Installation
Conda environment in environment.yml

`$ conda env create -n <environment name>    # default atari-rl`  
`$ conda activate <environment name> `  

### Training Atari 2600 Frogger
`$ python train.py`  

### Play existing models
`$ python play.py`
